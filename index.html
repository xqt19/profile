Trails App

The Trails App was a giant final project I had to complete for the bootcamp.
It is a Progressive Web Application(PWA) designed to be a one-stop shop for all your travel needs.

With Trails, you can plan trips with invited friends, and delegate items to bring for each other.
You can even view all your activities laid out neatly on a single map!



Lets Try It!

Trails is coded using Ruby on Rails and uses a heavy amount of AJAX and Javascript, backed up by other services like Mapbox API, ActiveRecord, and PostgreSQL. It is deployed on Herokuapp.
As this project was a team effort, We had much practice using Github and ActiveRecord Migration to control the workflow.

Trails is designed for use on a mobile phone.
Don't forget to add the app to homescreen so that you can see it in all its PWA glory!
